 # To-Do List with Priority & Deadlines

A simple **Python-based To-Do List application** that allows you to manage tasks with priority levels and deadlines. The program is fully interactive and works in terminal environments like **Termux**, making it lightweight and easy to run anywhere.

---

## **Features**

- Add tasks with **title, priority (High/Medium/Low), and deadline (YYYY-MM-DD)**.
- Show all tasks sorted by **priority and deadline**.
- Delete tasks by selecting the task number.
- Flexible menu: you can use **numbers** (1-4) or **text commands** like `add`, `show`, `delete`, or `exit`.
- Tasks are saved in a **JSON file (`tasks.json`)** for persistence.
- Fully terminal-friendly, no GUI required.
